# Sound Effects Reference

This directory should contain the following audio files (converted to MP3 format):

## Required Sound Effects

1. **correct_pop.mp3** - Correct word selection
   - Duration: ~200-300ms
   - Tone: Bright, positive pop sound
   - Volume: Normal

2. **incorrect_boop.mp3** - Incorrect word selection  
   - Duration: ~150-200ms
   - Tone: Soft, gentle "boop" sound
   - Volume: Quieter than correct

3. **completion_fanfare.mp3** - Sentence completion celebration
   - Duration: ~800-1200ms
   - Tone: Celebratory fanfare or chime sequence
   - Volume: Moderate-loud

4. **hint_chime.mp3** - Hint revealed
   - Duration: ~200-300ms
   - Tone: Soft, attention-getting chime
   - Volume: Normal

5. **success_ding.mp3** - General success indicator
   - Duration: ~200-300ms
   - Tone: Bright ding or bell sound
   - Volume: Normal

## Audio Guidelines

- All sounds should be kid-friendly and toy-like
- No harsh or scary tones
- Use simple instruments (bells, xylophones, etc.)
- All should be royalty-free or licensed for use
- Recommend using services like:
  - Freesound.org
  - Zapsplat.com
  - Pixabay Music
  - Bensound.com
